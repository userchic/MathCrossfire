# Перестрелка

Данный репозиторий является продолжением давно выполненной моей курсовой работы 2 курса. Сейчас проект доработан под стандарты репозитория MathBattles и также здесь используется SignalR
## Смысл игры

Игра состоит в том, что несколько команд по сколько то человек противостоят друг другу. Они решают задачи и при этом они могут указать цель, которая при правильном ответе будет с некоторой вероятностью поражена выстрелом. Выстрел отнимет некоторое количество очков целевой команды.

Игра может кончиться по времени либо когда все команды решат все задачи.

Игра по-моему аналогов не имеет.


## Отчёт
Сейчас в приложении:
- система разделена на фронтенд на React 19 и бекенд  WebApi на .Net 8
- на бекенде есть авторизация и аутентификация по кукам, репозитории(паттерн)
- логика в контроллерах, проект надо будет отрефакторить.
- контроллеров 3, это количество в некоторой степени основано на том, что я сделал в репозиории MathBattles. Есть контроллер аутентификации , который еще дает минимальную информацию об играх. Контроллер операций не относящихся к игровому процессу и командам. И контроллер в котором можно присоединяться к командам и отправлять ответы на задачи.
- хаб SignalR пока что занимается отправкой сообщений об отправленных игроками перестрелки ответах на задачи. Таким образом в интерфейсе перестрелки отображаются все ответы отправляемые игроками в реальном времени.

Для игрового процесса добавлены и разрабатываются: 
- Отображение статусов игроков в игре
- некоторые функции, которые пока что на фронтенде не реализованы и не являются важными(это уход из команды, возможно надо будет еще вернуть какие-то CRUD операции которые я убрал)
